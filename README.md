## Description
Personal Blog on Flask

### Supports 
* Comments
* Likes
* Share
* Filter posts by date
* Show related posts
	* Based on likes amount
* Profile editing
	* Avatar
	* Bio
* Privalage seperating
	* Confirmed users can comment/like posts
	* Another admin can't edit/delete the posts of another
* Post's time converting
